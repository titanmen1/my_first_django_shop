Интернет магазин на джанго 2.1
Функции:
	-Просмотр товара
	-Просмотр товара через категории
	-Добавление(удаление) товара в корзину
	-Динамическое обновление количества товара через jQuerry и Ajax
	-Изменение количества товара в корзине
	-Подтверждение заказ
	-Оформление заказ
	-Регистрация пользователей
	-Аунтетификация пользователей
	-Личный кабинет пользователя с просмотром подробностей заказа
	-Фильтрация заказов в панели администратора
